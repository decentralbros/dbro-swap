> Install dependencies using [pnpm](https://pnpm.io)

## `apps/web`

How to start

```sh
pnpm i
```

start the development server

```sh
pnpm dev
```

build with production mode

```sh
pnpm build

# start the application after build
pnpm start
```

## `apps/aptos`

How to start

```sh
pnpm dev:aptos
```

```sh
pnpm build:aptos
```

## `apps/blog`
